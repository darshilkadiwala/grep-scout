# FileScout has been renamed to GrepScout

> **⚠️ DEPRECATION NOTICE**
>
> The **FileScout** extension has been officially rebranded to **GrepScout**.
>
> This version (`filescout`) is now deprecated and exists purely to automatically install the new **GrepScout** extension for existing users to ensure a seamless transition.

## What do I need to do?

Nothing! If you are seeing this, VS Code should have already automatically installed the new **GrepScout** extension for you.

You can safely **Uninstall** this `filescout` extension from your VS Code editor.
